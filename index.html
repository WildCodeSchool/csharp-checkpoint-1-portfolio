üå† Finish with algorithms

Lucas would like you to show him how good you are with algorithmic conception. He gives you a set of problems to resolve with your own brain.

Here is what you should produce:

Sum of a values set
A function (a static method) that sums its first parameter which is an array of integers. The function returns the sum of the array elements. Example: [1,2,3,4] would return 10 (1+2+3+4 = 10)
Mean of a values set
A function (a static method) takes an array of integers as a parameter and returns the mean of an array. THe mean is returned by the function. Here is the formula to calculate a mean: (sum of values)/(number of values). Example: for [4,3,2,0,1], the method would return its mean which is 2 (10/5 = 2).
Sort
A function (a static method) takes an array of integers as its only parameter and sort it ascendently. The sorted array is returned by the function. Example: [4,3,2,1] would be sorted as [1,2,3,4]
